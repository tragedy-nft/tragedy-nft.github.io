# エフェクトアイデア集

## 実装済みエフェクト

- **bats** - コウモリが飛び回る
- **blackout** - 血の気が引いて視界が暗転（恐怖・衝撃表現）
- **blizzard** - 吹雪
- **bubble** - 泡
- **fire** - 炎
- **lightning** - 稲妻
- **matrix** - マトリックス風の緑の文字列
- **rainbow** - 虹
- **ripple** - 波紋
- **sparkle** - キラキラ
- **spinner** - 回転
